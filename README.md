# Proyecto Python - Portafolio

Este proyecto fue creado como ejemplo para cumplir con los **Requerimientos Funcionales Mínimos** de un módulo de programación en Python.

## Descripción general

El programa se ejecuta en consola y muestra un **menú principal** desde el cual se pueden probar distintas funcionalidades, cada una asociada a un requerimiento del curso.

## Funcionalidades incluidas

1. **Uso de conceptos fundamentales del lenguaje Python**
   - Conversor de unidades (temperatura y longitudes).
   - Uso de variables, operadores y tipos básicos (`int`, `float`, `str`).

2. **Manejo de tipos de datos y sentencias básicas**
   - Formulario en consola que solicita datos al usuario:
     - Nombre (cadena).
     - Edad (entero).
     - Estatura (float).
     - ¿Es estudiante? (booleano).

3. **Sentencias condicionales (`if`, `elif`, `else`)**
   - Análisis de un número para determinar si es:
     - Positivo.
     - Negativo.
     - Cero.

4. **Sentencias iterativas (`for` y `while`)**
   - Generador de tabla de multiplicar (uso de `for`).
   - Cálculo de factorial (uso de `while`).

5. **Estructuras de datos**
   - Agenda de contactos implementada con un **diccionario**:
     - Agregar, ver, buscar y eliminar contactos.

6. **Funciones para modularización y reutilización**
   - Cálculo de áreas de distintas figuras geométricas:
     - Círculo.
     - Rectángulo.
     - Triángulo.
   - Cada figura tiene su propia función:
     - `area_circulo`
     - `area_rectangulo`
     - `area_triangulo`

## Cómo ejecutar el proyecto

1. Asegúrate de tener instalado **Python 3**.
2. Descarga o descomprime la carpeta del proyecto.
3. Abre una terminal en la carpeta del proyecto.
4. Ejecuta:


python main.py

*(En algunos sistemas el comando puede ser `python3` en lugar de `python`.)*
